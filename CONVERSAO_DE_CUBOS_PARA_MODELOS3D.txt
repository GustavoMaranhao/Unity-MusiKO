Sob o NPC:

1. arrastar o player
2. mudar Mesh Filter do NPC para none
3. Escalar o player (0.1, 0.1, 0.1) por exemplo
   Mudar a Transform.Y do player para -0.5
4. Mudar o box collider para

  center  =  0.209541   =
  size    =  1.433694   =

5. Mudar Nav Mesh Agent

 Base Offset 0.25

6. Associar ao player o script iniciaPlayer
7. Associar ao player o material desejado
